# M9K Reloaded (M9KR) — Extra Weapons Pack for Garry's Mod

An expansion pack for the M9K Reloaded weapon base, adding 81 weapons spanning six categories. Includes a mix of modern military hardware, Cold War classics, sci-fi firearms, and exotic sidearms — all built on the M9KR base with full support for fire modes, caliber-driven ballistics, custom muzzle flashes, shell ejection, tactical reloads, and the rest of the M9KR feature set.

## Requirements

- [Garry's Mod](https://store.steampowered.com/app/4000/Garrys_Mod/)
- [M9K Reloaded Base](https://github.com/JlossBustin/M9K_Reloaded_Base)

## Features

All weapons inherit from the M9KR base and support:

- **Fire mode cycling** — semi-auto, burst, full-auto (per weapon)
- **Safety mode** — lowers weapon and blocks all input
- **Tactical & empty reloads** — chamber +1 system with distinct animations
- **Caliber-driven ballistics** — penetration, ricochet, and tracers based on real-world caliber data
- **Custom muzzle flashes** — PCF particle effects per weapon category
- **Shell ejection** — caliber-appropriate casings with physics and collision sounds
- **Suppressor system** — 4 weapons support in-game suppressor attach/detach (CAR-15, HK XM8, HK MK23, R5 RGP)
- **Scope overlays** — multiple optic types with animated sway (scoped weapons)
- **Custom HUD** — weapon name, fire mode, caliber, magazine count, and reserve ammo
